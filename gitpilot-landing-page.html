Frequently Asked Questions

Is GitPilot secure?

Absolutely. Security is our top priority. We use GitHub's official OAuth app for authentication, which means we never see or store your GitHub password. We only request the minimum permissions necessary to perform actions on your behalf and we never clone or store your source code.

What happens when a bulk action fails?

GitPilot is designed for resilience. If an action on a specific repository fails (e.g., due to permissions), the bulk job will continue with the rest of the repositories. You'll receive a detailed report of which actions succeeded and which failed, along with the error message from GitHub, so you can easily debug and retry.

Can I use this for my company's GitHub organization?

Yes! The Commander plan is built specifically for teams and organizations. You'll need to grant the GitPilot GitHub App access to your organization, and then you can perform bulk actions on any repositories you have administrative access to.
